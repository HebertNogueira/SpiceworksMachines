# SpiceworksMachines
Organizador simples de equipamentos encontrados pelo Spiceworks.

Direciona equipamentos para a pagina de informações em __www.dell.com__, sendo equipamentos Dell a página mostrará informações de garantia e peças.



__Ambiente de codificação__

* Debian 9.8

__Pacotes necessários__

* Apache, PHP, SQLite.



Copiar o arquivo __spiceworks_prod.db__ da pasta de instalação do Spiceworks e colar na pasta deste projeto.

Pasta de instalção padrão Spiceworks para cópia do arquivo: __Arquivos de Programas > Spiceworks > db__



Necessário habilitar __extension=php_pdo_sqlite.dll__ no __php.ini__.
